AI Assistant Widget Test Page

This page demonstrates how to embed the AI Assistant widget on your website. The chat button should appear in the bottom-right corner of this page.

Note: Make sure to run npm run build:widget first to generate the widget.js file.

How to Embed

Simply add the following script tag to your HTML:

<link rel="stylesheet" href="./dist/widget.css"> <script src="./dist/widget.iife.js"></script>

For production use, replace the paths with your CDN URLs:

<link rel="stylesheet" href="https://your-cdn.com/widget.css"> <script src="https://your-cdn.com/widget.iife.js"></script>

Features

  • ✅ Voice chat with AI assistant
  • ✅ Text chat interface
  • ✅ Floating button that expands on click
  • ✅ Fully self-contained - no additional dependencies needed
  • ✅ Works on any website
